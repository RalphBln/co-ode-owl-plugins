<h2>Beanshell View</h2>
<p>Get access to the P4 API directly through scripting.
The following variables are available:
<p><b>eKit</b> the OWLEditorKit for the current window
<p><b>mngr</b> the OWLModelManager for the current window

<p><a href="http://code.google.com/p/co-ode-owl-plugins/wiki/BeanshellView">Documentation</a>
